# 4. Mo rong prompting va chien luoc dieu huong

Trong qua trinh mo rong repo, mot huong cai tien quan trong la nang cap prompting va chien luoc dieu huong cua he thong agent. Neu chi dua vao mot prompt tong quat, model thuong co xu huong thao tac theo cach phan ung tinh huong, nghia la thay output nao thi xu ly output do, nhung khong giu duoc mot chien luoc giai bai on dinh trong nhieu vong. Dieu nay dac biet de thay o cac bai pwn, rev hoac web phuc tap, khi he thong phai vua phan tich file, vua kiem tra remote, vua thay doi huong suy luan theo ket qua moi. Vi vay, mo rong prompting khong chi la viet prompt dai hon, ma la thiet ke prompt theo vai tro, theo giai doan, va theo muc tieu cua tung agent trong toan bo pipeline.

Huong mo rong thu nhat la tach prompting theo vai tro chuyen biet. Trong repo hien tai da co su phan tach giua autoprompter, planner va executor, nhung co the day xa hon bang cach quy dinh ro hon nhiem vu nhan thuc cua tung agent. Auto-prompter nen duoc huan luyen de doc de bai, nhan dang loai challenge, tai nguyen co san, kieu loi co kha nang xuat hien va tu do sinh ra mot prompt dau vao sat voi challenge cu the thay vi mot prompt chung chung. Planner nen nhan prompt theo huong lap ke hoach kiem dinh gia thuyet: moi task giao cho executor phai gan voi mot gia thuyet co the xac nhan hoac bac bo. Executor nen nhan prompt theo huong hanh dong thao tac: uu tien lay fact moi, tranh lap lai lenh da thu, va neu mot huong that bai thi phai neu ro can cu de pivot. Cach tach nay giup he thong giam tinh mo ho va giam kha nang cac agent lap lai cung mot mau thought.

Huong mo rong thu hai la bo sung prompting theo giai doan giai bai. Thay vi de model tu do quyet dinh toan bo qua trinh, co the to chuc prompt thanh cac phase nhu: recon, xac dinh primitive, kiem chung gia thuyet, khai thac, va xac nhan flag. Trong phase recon, prompt nen ep agent uu tien thu thap thong tin co cau truc nhu file type, protection, symbol, endpoint, format input, va tinh kha dung cua remote. Trong phase xac dinh primitive, prompt nen yeu cau agent chi ro loi chinh la gi, vi du buffer overflow, format string, auth bypass, SQLi, hay crypto misuse. Trong phase kiem chung, prompt nen yeu cau agent dat mot thu nghiem nho de phan biet giua hai huong giai gan nhau. Trong phase exploit, prompt nen tap trung vao payload, script hoa, va thu local truoc remote neu co the. Cach chia phase nhu vay giup model co truc tu duy ro rang hon va de dua ra hanh dong tiep theo hop ly hon.

Huong mo rong thu ba la dua loop-control va tu phan bien vao prompting. Mot trong nhung ly do khien he thong that bai la model khong goi tool dung luc, hoac tiep tuc lap lai mot huong da khong con gia tri. De khac phuc, prompt can chen cac chi dan mang tinh dieu huong nhu: neu ba buoc gan nhat khong tao ra fact moi thi phai doi cach kiem tra; neu remote fail vi DNS hoac connection thi phai danh dau day la blocker cua environment thay vi tiep tuc viet exploit moi; neu decompile that bai thi phai chuyen sang strings, file, objdump, hoac source neu co. Ben canh do, co the them mot critic agent hoac mot critic prompt ngan de yeu cau he thong tu danh gia: gia thuyet nao vua bi bac bo, thong tin nao dang thieu, va buoc ke tiep nao thuc su khac voi buoc truoc. Cach lam nay giup prompt khong chi mo ta muc tieu, ma con dieu khien nhung luc model co nguy co bi ket.

Huong mo rong thu tu la su dung prompting theo tung mảng challenge. Mot prompt chung cho moi challenge thuong khong toi uu, vi crypto, pwn, web, rev va forensics co cach suy luan rat khac nhau. Voi pwn, prompt nen uu tien binary triage, mitigation, crash primitive, offset, local/remote parity va exploit script. Voi rev, prompt nen uu tien source recovery, strings, control-flow, check logic va encode/decode path. Voi web, prompt nen nhan manh trust boundary, session, role, input surface, va khac biet response. Voi crypto, prompt nen huong vao loai thuat toan, leakage, malformed parameter, va kha nang viet solver. Voi forensics, prompt nen dinh huong chuoi thao tac phan tich thay vi goi tool mot cach ngau nhien. Day la ly do repo nen duy tri he prompt rieng theo category va tiep tuc mo rong thanh prompt template chuyen biet hon nua.

Huong mo rong thu nam la ket hop prompting voi chien luoc dieu huong dua tren ket qua quan sat. Thay vi chi gui lai thong diep "Please run a command..." moi khi model khong goi tool, he thong nen dua them chi dan co dieu kien dua tren trang thai hien tai. Vi du, neu challenge co remote thi co the nhac agent uu tien kiem tra ket noi truoc khi viet exploit. Neu challenge co source file thi co the nhac agent doc source truoc khi decompile. Neu executor vua tra ve mot summary mo ho, planner co the duoc prompt de tach task hep hon, nhu "xac dinh chinh xac offset", "xac minh gadget pop rdi", hoac "kiem tra endpoint login co session fixation khong". Dieu nay bien prompting thanh mot co che dieu huong dong, khong con la mot doan van ban tinh duoc dat o dau cuoc hoi thoai.

Tong ket lai, mo rong prompting va chien luoc dieu huong la mot trong nhung huong nang cap co tac dong lon nhat doi voi repo. Neu lam tot, he thong se khong chi "noi hay hon", ma se ra quyet dinh tot hon, goi tool dung luc hon, giam loop, giam false fail do environment, va tang kha nang giu dung huong giai trong cac challenge dai hoi nhieu buoc. Ve mat do an, day cung la phan mo rong hop ly vi no khong doi hoi thay doi toan bo ha tang, nhung lai cai thien truc tiep chat luong suy luan va kha nang phoi hop giua cac agent.
